# WebView Media Launcher

## Описание проекта

**WebView Media Launcher** — это минималистичное Android-приложение, разработанное для предоставления удобного доступа к двум популярным медиа-сервисам: YouTube и "Кинопоиск" через встроенные WebView. Приложение ориентировано на долгосрочные сессии, сохранение состояния просмотра и простоту использования, обеспечивая нативный опыт без сложной интеграции API.

## Инструкция по сборке

Для сборки отладочной версии APK выполните следующую команду в корневой директории проекта:

```bash
./gradlew assembleDebug
```

После успешной сборки, отладочный APK будет расположен по пути `app/build/outputs/apk/debug/app-debug.apk`.

## Скачивание готового APK

Вы можете скачать готовые отладочные APK-файлы, собранные через CI/CD, в разделе [Actions на GitHub](https://github.com/YOUR_USERNAME/YOUR_REPO_NAME/actions) (замените `YOUR_USERNAME` и `YOUR_REPO_NAME` на актуальные).

## Легальность

**WebView Media Launcher** является исключительно приложением-браузером для доступа к общедоступным веб-сайтам (`m.youtube.com` и `www.kinopoisk.ru`). Приложение не обходит никакую рекламу, не использует сторонние API для парсинга контента и не нарушает условий использования упомянутых сервисов. Все взаимодействие происходит через официальные веб-интерфейсы этих сайтов, встроенные в WebView. Мы не размещаем собственную рекламу и не модифицируем контент сайтов.
